Trabajo: Agentes de IA (capturas y explicación)

Página estática para GitHub Pages con las capturas del proceso (instalación, configuración y ejecución del agente). GitHub Pages busca un index.html como archivo de entrada. [page:1]

Estructura recomendada en GitHub

Coloca este archivo como index.html en la raíz del repositorio, y crea la carpeta img/ con las imágenes: img/1.jpg, img/2.jpg, img/3.jpg, img/4.jpg. GitHub Pages publicará archivos estáticos y usará index.html como entrada. [page:1]

Nota: si activas GitHub Pages (Settings → Pages), puedes obtener un enlace público tipo https://usuario.github.io/repositorio/. [page:1]
